==============================================================
Calibrate a new Detector for EDS fit and quantification
==============================================================

This guide provides instructions on calibrating the Energy Dispersive Spectroscopy (EDS) quantification on a new detector and modifying the `XSp_calibs` file accordingly.

`AutoEMXSp` uses some pre-defined values for peak shapes and background fitting, which differ oin function of the employed detector and EDS parameters (current, voltage).
Hence, for accurate EDS spectral fitting and quantification, a few calibration steps are required for every different set of employed conditions.

This guide does not discuss standard Silicon Drift Detector (SDD) calibration of energy zero and gain. You'll find this information at `EDS Detector Calibration <../user/sdd_calib.html>`_.
