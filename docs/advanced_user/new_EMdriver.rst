===================================
Add new Electron Microscope Driver
===================================

This guide provides instructions on how to modify the `EM_driver` file to control a new microscope through its API.

Before you begin, please verify that the `EM_driver` required for your microscope brand is not already available at [CederGroupHub/AutoEMX](https://github.com/CederGroupHub/AutoEMX/tree/main/autoemx/EM_driver).
